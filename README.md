# Mini-App-Template
мини-апп для распознования животных

## Запуск

1. Клонирование репозитория
   ```bash
   git clone https://github.com/pavel-nikitin-2022/Mini-App-Template.git
   ```
2. Установка зависимостей
   ```bash
   npm install
   ```
3. Запуск проекта
   ```bash
   npm start
   ```
